# Gimnasio
Este es el repositorio para el proyecto de programación.

El programa es un gestor de entrenamientos que permite añadir, guardar, buscar, analizar y visualizar ejercicios realizados por el usuario.

La idea es que el programa actúe como un diario de ejercicios.

El usuario podrá:
- Añadir ejercicios (por ejemplo; correr 5km, hacer press banca, estirar 20min...)
- Guardarlos en una lista (ArrayList)
- Agruparlos por semana (HashMap)
- Buscar y filtrar los ejercicios. Filtrar por tipos: cardio, fuerza, flex. Ordenar por calorías, duración...
- Métodos de cálculo de calorías quemadas
- Guardar y cargar los entrenamientos (CSV, JSON)
- Usar la interfaz gráfica
